Recommended fonts for professional application letters (engineering):
- Lato
- Source Sans 3
- TeX Gyre Heros
- Avenir Next
- Helvetica Neue
- Arial
- Times New Roman (serif option)
- Futura (modern sans alternative)

Use these family names with the Makefile FONT flag, e.g.: FONT="Lato"
